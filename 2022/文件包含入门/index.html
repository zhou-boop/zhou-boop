特殊的 PHP 语言标记

常见的 PHP 语言标记有如下几种：

  • <?php phpinfo() ?>：如果只含有 PHP 代码， ?> 不允许存在。?> 隐含一个 ;，所以最后一句可以不用分号。
  • <script language=“php”>phpinfo()<script>。
  • <? ?>：需要修改 PHP 的 short_open_tag 配置
  • <% %>：默认禁用。
  • <?=$variablename ?>：默认可用。
  • <%=$variablename  %>：默认禁用。
